    Note for Trading

    Posted on 2025-07-26 In Finance Disqus:

    开始学习理财，从了解炒股入手。

    基础K线（Candlestick Charts）

    开盘价（Open）、收盘价（Close）、最高价（High）、最低价（Low）一目了然。

    反转形态（周线级别的反转形态更可靠）

    • 锤头线（Hammer） 下跌趋势底部看涨信号？
    • 上吊线（Hanging Man） 上涨趋势顶部看跌警示？
    • 吞没形态（Engulfing） 趋势反转的强信号？

    持续形态

    • 十字星（Doji）：市场犹豫，可能转向？
    • 三只乌鸦（Three Crows）：强烈看跌，跌势延续？

    简单移动平均线（SMA） (Weekly 50, 100, 200)

    给定周期内「收盘价」的平均值。

    SMA用法

    • 看趋势：股价相较SMA，在上方为上升、下方就是下降趋势。
    • 看支撑/阻力：周线SMA 50/100/200是长线关键防线，在下降途中视之为支撑、上涨途中视之为阻力。
    • 看金叉/死叉：短周期上穿长周期（金叉）看涨，下穿（死叉）看跌。？

    指数移动平均线（EMA） (8, 21)

    相比SMA，赋予近期价格更高权重。

    • EMA 8：超短期趋势，捕捉短线波动
    • EMA 21：中期趋势，俗称”操盘线”

    SMA v.s. EMA选择

    • 长线趋势判断：周线SMA 200
    • 短线交易信号：EMA 8/21
    • 消除市场噪音：SMA平滑效果较好，EMA易受短期波动影响

    一目均衡表 Ichimoku-cloud

    五大核心构成线

    • 转换线（Tenkan-sen）：9周期高低点均值 → 短期动能
    • 基准线（Kijun-sen）：26周期高低点均值 → 中期趋势强弱
    • 先行带A（Senkou Span A）：(转换线+基准线)/2 → 提前26期绘制，云图上沿
    • 先行带B（Senkou Span B）：52周期高低点均值 → 提前26期绘制，云图下沿
    • 迟行线（Chikou Span）：当前收盘价延后26期 → 验证趋势持续性

    用法

    • 厚度：云层越厚支撑/阻力越强
    • 颜色：绿色（看涨云），红色（看跌云）
    • 价格位置：
      • 云上 → 强势上涨趋势
      • 云中 → 震荡整理阶段
      • 云下 → 强烈下跌趋势

    交易信号

    • 三叉信号：转换线 > 基准线 + 价格在云上 + 迟行线高于26日前价格
    • 云层突破：价格上穿云顶，伴随成交量放大

    布林带 Bollinger band

    构成公式

    • 中轨 = 20日SMA
    • 上轨 = 20日SMA + 2倍标准差
    • 下轨 = 20日SMA - 2倍标准差

    用法

    • 波动率监测：通道收窄（收口）预示变盘在即，通道扩张（开口）标志趋势加速
    • 超买超卖：价格触及上轨→短期超买，触及下轨→短期超卖
    • 趋势跟踪：价格沿上轨运行=强势多头，沿下轨运行=强烈空头
    • 反转信号：”M顶”破中轨确认转跌，”W底”穿中轨确认转涨

    Fibonacci Retracement

    • 关键回撤位：23.6% → 38.2% → 50%（心理位）→ 61.8%（黄金分割）→ 78.6%
    • 上升趋势：从波段低点→高点画线
    • 下降趋势：从波段高点→低点画线
    • 有效画法：连接显著高低点（非随机波动点）
